# Contributing

Contributions should keep this project documentation-first and platform-agnostic.

## Accepted Contributions

- Better prompts.
- Better playlist schema fields.
- New platform adapter notes.
- Additional sample outputs clearly labeled by platform.
- Static HTML or EPUB publishing improvements.
- Open-license visual asset suggestions with manifest entries.
- Corrections to composer names, works, IPA, metadata, or historical framing.

## Not Accepted

- Platform automation scripts.
- API clients or executable code.
- Scraped album art, artist images, screenshots, or thumbnails without separate license review.
- Account-specific logs, cookies, browser profiles, or credentials.
- A sample that silently depends on one music platform.

## Style

- Keep `README.md` and `README.zh-CN.md` as separate language entry points.
- Keep core docs, specs, prompts, and templates in English unless a separate localized file is added.
- Keep prompts copy-ready.
- Prefer behavior-level instructions over implementation details.
- Cite official platform or asset-policy sources when adding rules.
