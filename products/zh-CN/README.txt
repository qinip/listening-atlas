古典音乐图谱：中文版指南包

这是 Classical Atlas demo 的 Apple Music edition。Apple Music 搜索列、面向 Apple Music 的执行说明，以及重建的平台执行日志，都是本 edition 的有意保留内容。可复用的平台无关 kit contract 请使用 ../../specs/、../../prompts/ 和 ../../templates/。

包含文件：
- classical_atlas_guide.zh-CN.md：中文 Markdown 指南
- classical_atlas_guide.zh-CN.html：中文静态 HTML 电子书
- classical_atlas_playlist.zh-CN.tsv：中文歌单数据
- classical_atlas_apple_music_execution_log.zh-CN.tsv：重建的 Apple Music case-study 执行日志
- create_playlist_on_platform.prompt.zh-CN.txt：用于 Apple Music edition 复现歌单的 computer-use prompt
- THIRD_PARTY_NOTICES.txt：第三方素材授权说明
- LICENSE.md：产品授权文本，说明 CC BY 4.0 demo 内容和 CC0 包内 prompt
- assets/：肖像图片，以及记录来源和授权信息的 ASSET_MANIFEST.tsv

原创指南文字、歌单表格、重建的 execution log 和 HTML 布局采用 CC BY 4.0。包内 prompt 采用 CC0 1.0。第三方链接、平台 metadata 和打包视觉素材保留各自原始或 manifest 记录的授权条款。

本包不包含平台自动化脚本，不包含专辑封面、平台截图或未进入 manifest 的图片。
